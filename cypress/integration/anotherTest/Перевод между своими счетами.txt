Перевод между своими счетами с одной корпоративной карты на другую

Предусловие
    - Произведена авторизация на сайте isfront.ru. Открыта главная страница сайта isfront.ru.

1. В нижней левой части главной страницы нажать на кнопку "Между своими счетами"
    - Открылась страница перевода между счетами 
2. Нажать на поле "Со счёта"
    - Открылось меню для выбора счета с которого будет произведен перевод 
3. В меню "Со счёта" выбрать счёт "Корпоративная карта" с которого будет произведен перевод
    - В поле "Со счёта" отображается выбраный счёт, меню закрыто 
4. Нажать на поле "Получатель"
    - Открылось меню для выбора получателя перевода
5. В меню "Получатель" выбрать "Информационные системы компания ООО" - получателя перевода
    - В поле "Получатель" отображается выбраный получатель, меню закрыто 
6. Нажать на поле "На счёт"
    - Открылось меню для выбора счета на который будет произведен перевод 
7. В меню "На счёт" выбрать "Корпоративная карта" - счёт на который будет произведен перевод
    - В поле "На счёт" отображается выбраный счет, меню закрыто 
8. Нажать на поле "Сумма перевода"
    - В поле "Сумма перевода" появился текстовый указатель
9. Ввести в поле "Сумма перевода" значение состоящее из цифр
    - В поле "Сумма перевода" отображается введенное значение 
10. Нажать на поле "Найти значение"
    - Открылся справочник назначений платежа 
11. В открытом справочнике нажать на выбранное назначение платежа 
    - Справочник закрыт, в поле "Назначение платежа" указано выбранное ранее назначение платежа
12. Нажать на кнопку "Далее" внизу от поля "Назначение платежа"
    - Открылось страница "Подтверждение операции"
13. В поле "Код из СМС" ввести значение "123456" 
    - В поле В поле "Код из СМС" отображается значение "123456" 
14. Нажать на кнопку "Подтвердить" внизу от поля "Код из СМС"
    - Открылась страница "Переводы", перевод по указанным в шагах 2-11 реквизитам совершен.
